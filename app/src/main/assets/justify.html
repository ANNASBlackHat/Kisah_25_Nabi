Last couple of days I've been working on open-sourcing my app Wear for Swarm. This application offers to make a Foursquare check-in using Android wearable device. I've been working on it since Android Wear Preview SDK was announced. With first release app was only looking for neaby venues regarding location change in certain time period and offered check-in using automatic system notifications. Last days I've added native support for check-in, so user can make a check-in explicitily (when he needs). Automatic notifications are still optional feature. It's possible to enable it in settings. I'll be happy for any feedback!
